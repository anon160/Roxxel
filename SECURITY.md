# Security Policy

## Supported Versions                                                                                                                                    

We only support security updates for the latest stable release of `roxxel`.

## Reporting a Vulnerability                                                                                                                             

**Please do not open public GitHub issues for security vulnerabilities.**

If you discover a security vulnerability in this project, please report it via **GitHub Private Vulnerability Reporting.**

### Acknowledgment
**We will acknowledge receipt of your report as soon as possible.**
